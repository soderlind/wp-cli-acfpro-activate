# Activate ACF PRO license using WP CLI

## OPTIONS

 `--key=<licensekey>`

 You'll find your ACF PRO license key at https://www.advancedcustomfields.com/my-account/

## EXAMPLE

 `wp acfpro activate --key=XXXXXX`
